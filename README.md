## Environment

We conduct experiments with Python 3.8.13 and Pytorch 1.13.0.

To setup the environment, please simply run

```
pip install -r requirements.txt
```

## Dataset

We use the Touch and Go dataset for training and testing. In order for you to start quickly, we provide the completed feature extraction files that you can directly [download](https://pan.quark.cn/s/585afb27e622) and place in the 'TV-CCL/data'.


## Quick Start
To quick start our TV-CCL, please run

```
sh train.sh
```
